# Maulana AI

**Maulana AI** adalah proyek chatbot AI sederhana yang dibuat menggunakan **HTML**, **CSS**, dan **JavaScript**, dengan integrasi ke API OpenAI.

## Fitur

- Chatbot AI menggunakan OpenAI API
- Antarmuka responsif dengan animasi keren
- Cocok untuk pembelajaran dan eksperimen AI frontend

## Teknologi

- HTML5
- CSS3 (dengan animasi)
- JavaScript (Fetch API)
- OpenAI API

## Cara Menjalankan

1. Clone repositori ini
2. Buka file `index.html` di browser
3. Masukkan API Key OpenAI kamu di bagian yang sudah disediakan di kode
